# 🧠 ZakiHub v3.0

&gt; منصة دليل أدوات الذكاء الاصطناعي الأكثر إبداعاً وإدماناً في 2026

## 🚀 خطوات النشر (5 خطوات فقط)

1. **أنشئ مستودع جديد على GitHub** (اسمه `zakihub`)
2. **ارفع كل الملفات** للمستودع
3. **فعّل GitHub Pages:**
   - ادخل على `Settings` → `Pages` → `Source` → اختار `GitHub Actions`
4. **فعّل GitHub Actions:**
   - ادخل على `Actions` → `I understand my workflows, go ahead and enable them`
5. **اضغط على زر `Run workflow`** في صفحة Actions

**خلاص! الموقع هيفضل شغال وحي لحاله.**
